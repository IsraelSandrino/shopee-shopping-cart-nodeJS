# 🛒 Carrinho de Compras — Shopee

Sistema de carrinho de compras executado no terminal, desenvolvido em Node.js como desafio de curso.

---

## 📋 Descrição

Simula a experiência de compra da Shopee via terminal, com menu interativo para gerenciar produtos no carrinho — sem necessidade de interface gráfica.

---

## ⚙️ Como instalar e rodar

**Pré-requisitos:** Node.js v18+

```bash
# Clone o repositório
git clone https://github.com/IsraelSandrino/carrinho-compras-shopee.git

# Acesse a pasta
cd carrinho-compras-shopee

# Instale as dependências
npm install

# Rode o projeto
npm start
```

---

## ✅ Funcionalidades

- Listar produtos do catálogo
- Adicionar produto ao carrinho com quantidade
- Remover uma unidade do produto
- Deletar produto do carrinho independente da quantidade
- Visualizar carrinho com subtotal por item
- Calcular total da compra em BRL

---

## 🚀 Tecnologias

- [Node.js](https://nodejs.org/) — runtime JavaScript
- `readline` — módulo nativo para input no terminal
- ES Modules — sistema de módulos nativo do Node.js
